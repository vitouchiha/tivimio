Endpoint Principali

🔄 Proxy Universale

GET /proxy?url=<STREAM_URL> Supporta: • HLS (.m3u8) - Streaming live e VOD • M3U playlist - Liste canali IPTV • MPD (DASH) - Streaming adattivo • PHP streams - Flussi dinamici Esempio: /proxy?url=https://example.com/stream.m3u8

🎬 Estrazione Vavoo

GET /vavoo?url=<URL_VAVOO> Risolve automaticamente: • Link vavoo.to in stream diretti • Autenticazione API automatica • Headers ottimizzati per streaming

📡 Risoluzione DaddyLive HD

GET /dlhd?url=<URL_DADDYLIVE> Funzionalità: • Risoluzione link DaddyLive HD • Bypass automatico restrizioni • Ottimizzazione qualità stream

Caratteristiche Avanzate

  • Gestione automatica AES-128 per chiavi crittografiche
  • Supporto completo #EXTVLCOPT e #EXTHTTP
  • Retry automatico con exponential backoff
  • Connection pooling per prestazioni ottimali
  • Headers personalizzati con prefisso h_
  • Proxy trasparente per segmenti .ts
  • Compatibilità universale player IPTV
  • Sessioni persistenti anti-bot
HLS M3U8 MPD PHP Vavoo DLHD AES-128

Headers Personalizzati

Aggiungi headers custom con prefisso h_:

?url=STREAM_URL&h_user-agent=CustomUA&h_referer=https://site.com&h_authorization=Bearer token123 Headers supportati: • h_user-agent - User Agent personalizzato • h_referer - Sito di riferimento • h_authorization - Token di autorizzazione • h_origin - Dominio origine • h_* - Qualsiasi header personalizzato
